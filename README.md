# المنظمة غير الحكومية EGAS | EGAS NGO

تم تصميم وتطوير هذا الموقع ليكون البوابة الرسمية لمنظمة EGAS غير الحكومية في زوارة، ليبيا، مع توفير تجربة مستخدم متعددة اللغات لدعم اللغة العربية، الانجليزية، والأمازيغية.

## 🌟 المميزات
- تصميم وتطوير متجاوب مع جميع أحجام الشاشات.
- دعم كامل لثلاث لغات (العربية، الإنجليزية، والأمازيغية - تيفيناغ).
- لوحة تحكم للوصول الديناميكي لآخر الأخبار باستخدام `localStorage`.
- هيكلية ملفات منظمة وسهلة التعديل تدعم الرفع المباشر عبر Netlify / GitHub Pages.

## 🛠️ التقنيات المستخدمة
- **HTML5** لهيكلة الصفحات وعناصرها.
- **CSS3** للتصميم وتنسيق الألوان وتجاوب الشاشات.
- **JavaScript** (Vanilla) للتفاعل، التحويل بين اللغات، وتمرير البيانات.

## 📂 هيكل المشروع
- `ar/` ، `en/` ، `tmz/`: مجلدات اللغات الثلاث وكل مسار يحتوي على صفحاته المخصصة.
- `css/`: ملفات التنسيق CSS.
- `js/`: الأكواد البرمجية والتفاعل للواجهة.
- `img/` و `fonts/`: الصور والأيقونات والخطوط المستخدمة.
- `admin/`: لوحة التحكم البسيطة لمقالات الموقع وأخباره.
